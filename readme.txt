Репозиторий с программой circles для поиска пересечений произвольного кол-ва окружностей. В папке bin содержатся исполняемые файлы circles и test. Папка build используется для хранения артефактов сборки. Скрипты .sh используются для работы с travis. Makefile собирает приложения в каталоге bin при применении в консоль команды make. Для удаления артефактов сборки из build вводится make clean. Для сборки теста - make test. Файл .travis.yml содержит информацию для travis. В библиотеке ctest.h реализовано сравнение двух чисел float. В случае несоотвествия чисел вызывается функция abort, которая аварийно останавливает программу.Почему-то после одного из коммитов travis перестал обновляться и стал лишь прогонять только один старый коммит.
